# Minimalist Timetable App

A simple, lightweight timetable application built with **PocketBase**, **Alpine.js**, and **Tailwind CSS**. Designed to work seamlessly on both Desktop and Mobile (PWA-ready).

## Features
- **Fast & Light:** Single-binary backend with PocketBase.
- **Cross-Platform:** Responsive UI that can be "Added to Home Screen" on Android.
- **Real-time:** Updates across devices instantly.

## Setup & Installation

### 1. Prerequisites
- Download the **PocketBase** binary for your OS from [pocketbase.io](https://pocketbase.io/).
- Place the `pocketbase` executable in the root of this project.
